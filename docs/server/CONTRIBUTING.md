# Contributing to Stricture Server

This document applies specifically to code under the Stricture Server surface,
including:

- `cmd/stricture-server/**`
- `internal/server/**`
- `docs/server/**`

## License

## Contribution Terms

By submitting any Stricture Server contribution (code, tests, docs, scripts, or
other material), you agree that:

1. You have the right to submit the contribution.
2. You assign all right, title, and interest in that contribution to **Brenn
   Hill**.
3. You consent to maintainers relicense or dual-license contributed material as
   needed for the project.

If you do not agree to these terms, do not submit Stricture Server
contributions.
